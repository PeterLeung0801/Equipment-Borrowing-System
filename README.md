# Equipment Borrowing System

**Author**: Leung Kam Chung Peter  
**Course**: COMP8090SEF  

This repo has two tasks:

- Task 1: OOP app for borrowing equipment
- Task 2: Heap and heap sort study

## Task 1

A system to manage equipment borrowing in labs.

Features:
- Add borrow records
- Mark as returned
- Delete records
- Show sorted list
- Stats and alerts
- Saves to JSON

Run: `python Task1/main.py` or `python Task1/gui.py`

## Task 2

Heap data structure and heap sort algorithm.

Run: `python Task2/heap_sort.py`

---

## Task 2 – Self Study

### Selected Data Structure
Max Heap

### Selected Algorithm
Heap Sort

The Heap data structure and Heap Sort algorithm are implemented and analyzed, including time complexity and practical applications.

### How to Run Task 2

`python Task2/heap_sort.py`
